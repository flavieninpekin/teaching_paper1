# 金融学模块 — 三卷本课程

## 整体设计

定位：AP / A-Level 至大一金融专业先修水平。三卷共24课。

## 学习路径

```
第一卷：金融学基础（8课）
    ↓
第二卷：衍生品与风险管理（6课）
    ↓
第三卷：金融创新与系统性风险（10课）
```

## 每课结构

1. **情景引入** — 一个真实事件或场景
2. **定性分析** — 直觉、框架、逻辑
3. **定量分析** — 数学模型、公式推导、计算
4. **总结** — 核心结论
5. **讨论题** — 开放性问题，无标准答案
6. **课后练习** — 计算题 + 分析题
7. **学海拾遗** — 学术细节、历史花絮、扩展阅读（侧栏）
