# 扩展知识模块 · 整体框架与写作规范

## 模块定位

面向小学高年级至高中学生（G5-G12），用故事引入各专业领域的核心概念，再用正式课程深入知识体系。每个模块仿照自然科学系列（nature-intro → physics/chemistry/biology/geography）的"故事入门→正课深化"结构，但增加了"多层次纵深"（intro → vol1 → vol2 → vol3）。

## 每个模块的标准结构

```
E04-finance/
├── README.md                 ← 模块说明
├── syllabus.tex              ← 完整大纲（列出intro + 各卷内容）
├── intro.tex                 ← 入门故事（G5-G8，故事化，每课配插图）
│                              每课结构：故事→想一想→有趣的小知识
├── vol1-foundations.tex      ← 第一卷·基础（G10-G12，情景→定性→定量→练习）
├── vol2-xxx.tex              ← 第二卷·专业方向（可选，视领域深度）
├── vol3-xxx.tex              ← 第三卷·前沿与伦理（可选，视领域深度）
└── figure/requirements/      ← 插图需求清单
```

## 每课统一结构（正课，vol1/2/3）

每课依次包含7个板块：

1. **情景引入**（Scenario）——一个真实事件、经典案例或贴近北京生活的场景，用一段叙述抓住读者的兴趣
2. **定性分析**（Qualitative）——解释核心概念的直观含义和逻辑框架，不出现公式
3. **定量分析**（Quantitative）——数学模型、公式推导和计算示例。公式不是目的，而是理解概念的工具
4. **总结**（Summary）——3-5条要点，凝练这课的核心结论
5. **讨论题**（Discussion）——开放性问题，没有标准答案，鼓励批判性思维
6. **课后练习**（Exercises）——计算题+分析题+探究题（可用于家庭作业或课堂讨论）
7. **学海拾遗**（Sidebar）——学术花絮、历史背景、扩展阅读、补充推导，不要求掌握但提供深度

## LaTeX 编写规范

### 样式包

每个正课文件（vol1/2/3）导言区引入：

```latex
% 金融学使用统一样式包（其他模块初期可自行定义，待全部模块统一时考虑公共样式包）
\usepackage{finance-style}
```

### 环境命令

| 环境 | 功能 | 颜色 |
|------|------|------|
| `scenario` | 情景引入 | 深红框 |
| `qualitative` | 定性分析 | 蓝色框 |
| `quantitative` | 定量分析 | 绿色框 |
| `lessonsummary` | 总结 | 绿色底 |
| `discuss` | 讨论题 | 橙色底 |
| `exercises` | 课后练习 | 蓝色底 |
| `figplaceholder{}` | 插图占位符 | 灰色框 |
| `xuehaiinline{}` | 学海拾遗（正文内嵌） | 天蓝边 |

### 插图需求

每个模块的 `figure/requirements/` 下存放插图需求清单（.txt 格式），格式仿照已有模块：

```
[H] 第二节 "量化交易" 段落末尾
  → 量化交易流程图
  → 五步流程：市场数据输入 → 策略模型分析 → 交易信号生成 → 自动下单执行 → 盈亏反馈
  → prompt: 详细描述用于 AI 绘图
```

优先级 H=高 M=中 L=低。
