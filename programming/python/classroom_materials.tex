\section*{\textcolor{blue!70!black}{说明}}

Python 是当前全球最流行的入门编程语言之一。它的语法简洁、接近自然语言，适合作为从 Scratch 图形化编程到文本编程的第一步。

\textbf{主要内容（初拟）：}
\begin{enumerate}
  \item Python 环境搭建与第一个程序
  \item 变量与数据类型（整数、浮点数、字符串、布尔值）
  \item 条件判断（\texttt{if}/\texttt{elif}/\texttt{else}）
  \item 循环（\texttt{for}、\texttt{while}）
  \item 列表、元组、字典
  \item 函数与模块
  \item 文件读写
  \item 小项目：猜数字、计算器、简易记账本
\end{enumerate}

\textbf{与Scratch的对照：}
\begin{itemize}
  \item 变量 $\leftrightarrow$ Scratch 中的变量积木
  \item \texttt{if} $\leftrightarrow$ Scratch 中的"如果……那么"积木
  \item \texttt{for}循环 $\leftrightarrow$ Scratch 中的"重复……次"积木
  \item 函数 $\leftrightarrow$ Scratch 中的"自制积木"
\end{itemize}
