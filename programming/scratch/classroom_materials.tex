\section*{\textcolor{blue!70!black}{什么是Scratch？}}

Scratch 是由美国麻省理工学院（MIT）媒体实验室开发的一款\textbf{免费图形化编程工具}，专门面向 8 到 16 岁的青少年。它于 2007 年首次发布，2019 年推出了 Scratch 3.0 版本，全面支持浏览器在线编辑和平板设备。

\textbf{它最大的特点是不用写代码。} 传统的编程需要记住英文单词和复杂的语法规则，比如告诉计算机"如果……就……"要用 \texttt{if} 和冒号。但在 Scratch 里，你只需要像搭积木一样，把彩色的指令块拖到编程区拼在一起就可以了。这些积木块有不同的颜色和形状，拼不上的就说明逻辑不对——这就把注意力从"记单词"转移到了"想逻辑"上。

\vspace{0.3cm}
\textbf{Scratch的核心概念：}

\begin{itemize}
  \item \textbf{角色（Sprite）}——程序里的"演员"，可以是一只小猫、一个球、或者任何你画的图案。
  \item \textbf{舞台（Stage）}——角色表演的地方，相当于一个剧场。
  \item \textbf{事件（Event）}——程序什么时候开始？点击绿旗？按下键盘？还是碰到另一个角色？
  \item \textbf{循环（Loop）}——让某段指令反复执行，比如"一直走，碰到边缘就反弹"。
  \item \textbf{条件判断（Conditional）}——如果……就……，比如"如果碰到了敌人，就消失"。
  \item \textbf{变量（Variable）}——用来存数或文字的小盒子，比如记录你的得分。
\end{itemize}

\vspace{0.3cm}
\textbf{为什么要先学Scratch？}

因为编程最核心的东西——顺序、分支、循环、变量、事件——在 Scratch 里全都能学到，而且完全不用操心语法错误。等这些概念都掌握了，再切换到 Python 这样的文本语言，区别仅仅是"从拖积木变成敲代码"，底层的思维方式是一样的。全球有数千万学生把 Scratch 作为编程启蒙的第一步。

\vspace{0.3cm}
\textbf{可以做什么？}

\begin{itemize}
  \item 制作动画故事（让小猫说话、走路、变魔术）
  \item 制作小游戏（接苹果、打地鼠、迷宫）
  \item 制作互动艺术（随鼠标变色的画）
  \item 甚至做一些简单的物理模拟
\end{itemize}

\textbf{学习路径建议：} Scratch ($\rightarrow$) Python ($\rightarrow$) C++。
