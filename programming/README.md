# programming/ —— 编程系列 · 六年完整课程大纲

> 面向北京地区四年级起点学生，G4–G9 六年编程素养培养
> 从 Scratch 图形化启蒙到独立交付完整项目

---

## 一、设计理念

### 1.1 核心理念

| 原则 | 说明 |
|---|---|
| **先思维，后语法** | Scratch → 计算思维 → Python，先建立编程直觉再学文本语法 |
| **逐层递加认知维度** | 每年增加一个新维度，不跳跃：图形化 → 思维 → 文本 → 算法 → 多语言 → 工程 → AI |
| **产出驱动** | 每个阶段都有可展示作品：游戏、网页、项目，保持成就感 |
| **家长可参与** | Scratch 模块专为家长编写，不懂编程也能理解和支持 |
| **对标主流** | 覆盖信息学奥赛入门要求、高中信息技术新课标核心内容 |

### 1.2 整体路径

```
G4 ──── G5 ──── G6 ──── G7 ──── G8 ──── G9
│       │       │       │       │       │
Scratch  计算思维  Python  C++     Web     AI
  │       │       │   │   │       │       │
  │       └─ 计算机数学 ─┘   │     Git ───┘
  │               │          │       │
  │          数据结构与算法 ──┘   综合项目
  └────────── 家长指南 ────────┘
```

### 1.3 模块依赖关系

```
scratch/  (无前置)
   ↓
computational-thinking/  (前置: scratch)
   ↓
computer_maths/  (可与 Python 并行)
   ↓
python/  (前置: computational-thinking)
   ↓
ds-algo/  (前置: python, 可参考 computer_maths)
   ↓   ↘
python-to-cpp/  (前置: python, 参考 ds-algo)
   ↓
web-dev/  (前置: python)
   ↓
git-collab/  (前置: web-dev, 实操对象)
   ↓
ai-intro/  (前置: python, 参考 computer_maths + ds-algo)
   ↓
capstone/  (前置: 全部)
```

## 三、各模块详细大纲

### 3.1 scratch/ —— Scratch 家长指南

**定位：** 写给不懂编程的家长，孩子已学 Scratch 多年。

| 章节 | 内容 |
|---|---|
| 一、Scratch 是什么 | MIT 开发，免费图形化编程，不用写代码 |
| 二、界面长什么样 | 舞台/角色列表/积木区/脚本区/绿旗 |
| 三、积木块颜色 | 9 类颜色各代表什么功能 |
| 四、孩子学了哪些编程概念 | 顺序执行/事件/循环/条件/变量/并行/广播/坐标 |
| 五、能做哪些项目 | 动画/游戏/互动艺术/模拟器 |
| 六、能力评估 | 5 层：模仿→修改→创作→调试→抽象 |
| 七、家长提问指南 | 看作品/聊学习时的问题话术 |
| 八、怎么看到作品 | 在线/离线，建议定期作品展示 |
| 九、Scratch → Python | 概念对照表 |
| 十、亲子练习 | 5 个亲子互动练习建议 |
| 关联模块 | 计算思维、计算机数学、Python、ds-algo |

### 3.2 computational-thinking/ —— 计算思维与计算机基础

**定位：** Scratch 到 Python 的桥梁，G4–G5。

| 课 | 标题 | 核心内容 |
|---|---|---|
| 1 | 什么是计算思维 | 分解/模式识别/抽象/算法（西红柿炒鸡蛋例子） |
| 2 | 分解 | 大事化小（接水果游戏拆解） |
| 3 | 模式识别 | 找规律（Scratch 游戏共同模板） |
| 4 | 抽象 | 抓住重点丢掉细节 |
| 5 | 算法设计 | 猜数字/二分查找 |
| 6 | 流程图 | 符号/实例（是否带伞） |
| 7 | 伪代码 | 猜数字伪代码 |
| 8 | 计算机怎么工作 | 二进制/CPU/内存/硬盘 |
| 9 | 程序怎么运行 | 编译 vs 解释 |
| 10 | 调试思维 | 调试四步法/bug类型 |

**练习：** 5 道基础题 + 3 道参考答案 + 扩展项目"自动售货机"

### 3.5 ds-algo/ —— 数据结构与算法初步

**定位：** 编程"内功"，竞赛基础，G6。

| 课 | 标题 | 复杂度 |
|---|---|---|
| 1 | 时间复杂度 | 大O对比表 |
| 2 | 数组 vs 链表 | O(1) vs O(n) 访问 |
| 3 | 栈与队列 | 括号匹配 |
| 4 | 树 | 二叉树遍历/BST |
| 5 | 图 | BFS/DFS |
| 6 | 排序（5种） | O(n²) vs O(n log n) |
| 7 | 二分查找 | O(log n) |
| 8 | 递归 | 阶乘/斐波那契 |
| 9 | 动态规划 | 爬楼梯/背包问题 |

**扩展项目：** 表达式计算器、文件搜索工具、排序性能对比

## 五、练习与项目体系

每个模块末尾包含三层练习结构：

| 层次 | 说明 | 覆盖模块 |
|---|---|---|
| **基础练习** | 每模块 5 道，含完整参考答案 | 全部 |
| **扩展项目** | 每模块 2-3 个，标注关联模块 | 全部 |
| **跨模块项目** | capstone 提供 7 个选题及 6 周计划 | capstone 汇总 |

### 典型跨模块项目

| 项目 | 涉及模块 | 综合度 |

| AI 图片分类网站 | python + web-dev + ai-intro + git-collab | ★★★★★ |
| 在线代码评判系统 | python + ds-algo + web-dev + git-collab | ★★★★★ |
| 背单词 App + 记忆曲线 | python + ds-algo + web-dev + computer_maths | ★★★★☆ |

| 算法可视化工具 | python + ds-algo + web-dev | ★★★☆☆ |
