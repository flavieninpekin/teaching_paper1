\section*{写在前面}

到目前为止，你写的程序都在命令行里运行——黑底白字，只有你能看见。

Web 开发让你\textbf{做出真正的产品}——任何人都能通过浏览器访问的网站。这是很多程序员爱上编程的原因：你写的东西，全世界都能用。

我们会用 4 个小项目，一步步从纯前端到全栈，做出一个完整的网站。

\subsection*{你需要准备}

\begin{itemize}
  \item 一个文本编辑器（推荐 VS Code：\url{https://code.visualstudio.com/}）
  \item Python 已安装（我们用它做后端）
  \item 一个浏览器（Chrome 或 Edge）
\end{itemize}

\section*{第一课：Web 是怎么工作的？}

\subsection*{浏览器 ↔ 服务器}

\begin{verbatim}
你的电脑                         服务器
┌─────────┐    发起请求（URL）    ┌─────────┐
│ 浏览器  │ ──────────────────→  │ 服务器  │
│         │    返回网页（HTML）   │         │
│         │ ←──────────────────  │         │
└─────────┘                      └─────────┘
\end{verbatim}

\begin{enumerate}
  \item 你在浏览器里输入 \texttt{www.example.com}
  \item 浏览器向 example.com 的服务器发送请求
  \item 服务器处理请求，返回 HTML 文件
  \item 浏览器把 HTML 渲染成你看到的页面
\end{enumerate}

\subsection*{前端 vs 后端 vs 数据库}

\begin{center}
\begin{tabular}{|c|l|}
\hline
\textbf{角色} & \textbf{干啥的} \\ \hline
前端 & 页面长什么样？按钮按了怎么动？——HTML + CSS + JavaScript \\ \hline
后端 & 数据怎么处理？用户登录对不对？——Python + Flask \\ \hline
数据库 & 数据存哪儿？用户信息、文章内容——SQLite \\ \hline
\end{tabular}
\end{center}

\section*{项目一：个人主页（纯前端）}

\subsection*{HTML——网页的骨架}

HTML 不是编程语言，它是一种\textbf{标记语言}——用标签告诉浏览器"这里是个标题"、"这里是个段落"。

\begin{lstlisting}[language=HTML]
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>我的个人主页</title>
</head>
<body>
    <h1>你好，我是小明！</h1>
    <p>我今年 12 岁，喜欢编程和打篮球。</p>
    
    <h2>我的爱好</h2>
    <ul>
        <li>编程（Scratch + Python）</li>
        <li>打篮球</li>
        <li>读科幻小说</li>
    </ul>
    
    <h2>我的作品</h2>
    <ol>
        <li>猜数字游戏（Python）</li>
        <li>简易记账本</li>
        <li>石头剪刀布</li>
    </ol>
    
    <a href="https://scratch.mit.edu">我的 Scratch 主页</a>
</body>
</html>
\end{lstlisting}

保存为 \texttt{index.html}，双击用浏览器打开——你的第一个网页！

\subsection*{CSS——网页的皮肤}

HTML 决定了内容，CSS 决定内容\textbf{长什么样}。

把样式写在 \texttt{<head>} 里的 \texttt{<style>} 标签中：

\begin{lstlisting}[language=HTML]
<head>
    <style>
        body {
            font-family: "微软雅黑", sans-serif;
            background-color: #f5f5f5;
            margin: 40px;
        }
        h1 {
            color: #2c3e50;
            text-align: center;
        }
        .card {
            background-color: white;
            padding: 20px;
            border-radius: 10px;
            box-shadow: 0 2px 5px rgba(0,0,0,0.1);
            margin-bottom: 20px;
        }
        a {
            color: #3498db;
            text-decoration: none;
        }
        a:hover {
            text-decoration: underline;
        }
    </style>
</head>
<body>
    <div class="card">
        <h1>你好，我是小明！</h1>
        <p>我今年 12 岁，喜欢编程和打篮球。</p>
    </div>
</body>
\end{lstlisting}

CSS 规则：\texttt{选择器 \{ 属性: 值; \}}

\begin{itemize}
  \item \texttt{h1} —— 选中所有 h1 标签
  \item \texttt{.card} —— 选中所有 class="card" 的元素（前面有点号）
  \item \texttt{\#id} —— 选中 id="xxx" 的元素（前面有井号）
\end{itemize}

\subsection*{使用 HTML 模板}

把网页内容写在独立的 HTML 文件中。在项目文件夹下创建 \texttt{templates/} 目录。

\texttt{templates/index.html:}

\begin{lstlisting}[language=HTML]
<!DOCTYPE html>
<html>
<head>
    <title>留言板</title>
    <style>
        body { font-family: "微软雅黑"; margin: 40px; background: #f0f0f0; }
        .message { background: white; padding: 15px; border-radius: 8px; margin: 10px 0; }
        form { background: white; padding: 20px; border-radius: 8px; }
        input, textarea { width: 100%; padding: 8px; margin: 5px 0; }
    </style>
</head>
<body>
    <h1>留言板</h1>
    
    <form action="/add" method="post">
        <input type="text" name="username" placeholder="你的名字" required>
        <textarea name="content" rows="3" placeholder="留言内容" required></textarea>
        <button type="submit">提交留言</button>
    </form>
    
    <h2>已有留言</h2>
    {% for msg in messages %}
    <div class="message">
        <strong>{{ msg.username }}</strong>
        <p>{{ msg.content }}</p>
    </div>
    {% endfor %}
</body>
</html>
\end{lstlisting}

\texttt{app.py:}

\begin{lstlisting}[language=Python]
from flask import Flask, render_template, request, redirect
import sqlite3
import os

app = Flask(__name__)

def init_db():
    conn = sqlite3.connect("messages.db")
    c = conn.cursor()
    c.execute("""
        CREATE TABLE IF NOT EXISTS messages (
            id INTEGER PRIMARY KEY AUTOINCREMENT,
            username TEXT NOT NULL,
            content TEXT NOT NULL
        )
    """)
    conn.commit()
    conn.close()

def get_messages():
    conn = sqlite3.connect("messages.db")
    c = conn.cursor()
    c.execute("SELECT username, content FROM messages ORDER BY id DESC")
    rows = c.fetchall()
    conn.close()
    return [{"username": r[0], "content": r[1]} for r in rows]

def add_message(username, content):
    conn = sqlite3.connect("messages.db")
    c = conn.cursor()
    c.execute("INSERT INTO messages (username, content) VALUES (?, ?)",
              (username, content))
    conn.commit()
    conn.close()

@app.route("/")
def index():
    messages = get_messages()
    return render_template("index.html", messages=messages)

@app.route("/add", methods=["POST"])
def add():
    username = request.form["username"]
    content = request.form["content"]
    add_message(username, content)
    return redirect("/")

if __name__ == "__main__":
    init_db()
    app.run(debug=True)
\end{lstlisting}

\textbf{运行流程：}
\begin{enumerate}
  \item 访问首页 → 从数据库读取所有留言 → 渲染 HTML 模板 → 显示在浏览器
  \item 填写表单提交 → POST 请求到 /add → 写入数据库 → 重定向回首页
\end{enumerate}

\subsection*{SQLite 是什么？}

SQLite 是一种轻量级数据库。它不需要安装服务器，数据存在一个文件（\texttt{messages.db}）里。

\begin{lstlisting}[language=SQL]
-- 创建表
CREATE TABLE messages (id INTEGER, username TEXT, content TEXT);

-- 插入数据
INSERT INTO messages (username, content) VALUES ('小明', '大家好！');

-- 查询数据
SELECT username, content FROM messages ORDER BY id DESC;
\end{lstlisting}

\subsection*{项目结构}

\begin{verbatim}

\subsection*{app.py 核心代码}

\begin{lstlisting}[language=Python]
from flask import Flask, render_template, request, redirect, session
import sqlite3
from functools import wraps

app = Flask(__name__)
app.secret_key = "your-secret-key"  # 用于加密 session

def get_db():
    conn = sqlite3.connect("blog.db")
    conn.row_factory = sqlite3.Row
    return conn

def login_required(f):
    @wraps(f)
    def decorated_function(*args, **kwargs):
        if "user_id" not in session:
            return redirect("/login")
        return f(*args, **kwargs)
    return decorated_function

@app.route("/")
def index():
    conn = get_db()
    posts = conn.execute("""
        SELECT posts.*, users.username 
        FROM posts JOIN users ON posts.user_id = users.id
        ORDER BY posts.created_at DESC
    """).fetchall()
    conn.close()
    return render_template("index.html", posts=posts)

@app.route("/register", methods=["GET", "POST"])
def register():
    if request.method == "POST":
        username = request.form["username"]
        password = request.form["password"]
        conn = get_db()
        try:
            conn.execute("INSERT INTO users (username, password) VALUES (?, ?)",
                        (username, password))
            conn.commit()
        except sqlite3.IntegrityError:
            return "用户名已存在"
        finally:
            conn.close()
        return redirect("/login")
    return render_template("register.html")

@app.route("/login", methods=["GET", "POST"])
def login():
    if request.method == "POST":
        username = request.form["username"]
        password = request.form["password"]
        conn = get_db()
        user = conn.execute("SELECT * FROM users WHERE username=? AND password=?",
                          (username, password)).fetchone()
        conn.close()
        if user:
            session["user_id"] = user["id"]
            session["username"] = user["username"]
            return redirect("/")
        return "用户名或密码错误"
    return render_template("login.html")

@app.route("/logout")
def logout():
    session.clear()
    return redirect("/")

@app.route("/create", methods=["GET", "POST"])
@login_required
def create():
    if request.method == "POST":
        title = request.form["title"]
        content = request.form["content"]
        conn = get_db()
        conn.execute("INSERT INTO posts (title, content, user_id) VALUES (?, ?, ?)",
                    (title, content, session["user_id"]))
        conn.commit()
        conn.close()
        return redirect("/")
    return render_template("create.html")

@app.route("/post/<int:post_id>")
def view_post(post_id):
    conn = get_db()
    post = conn.execute("""
        SELECT posts.*, users.username 
        FROM posts JOIN users ON posts.user_id = users.id
        WHERE posts.id=?
    """, (post_id,)).fetchone()
    
    comments = conn.execute("""
        SELECT comments.*, users.username
        FROM comments JOIN users ON comments.user_id = users.id
        WHERE comments.post_id=?
        ORDER BY comments.created_at
    """, (post_id,)).fetchall()
    conn.close()
    return render_template("post.html", post=post, comments=comments)

@app.route("/comment/<int:post_id>", methods=["POST"])
@login_required
def comment(post_id):
    content = request.form["content"]
    conn = get_db()
    conn.execute("INSERT INTO comments (post_id, user_id, content) VALUES (?, ?, ?)",
                (post_id, session["user_id"], content))
    conn.commit()
    conn.close()
    return redirect(f"/post/{post_id}")
\end{lstlisting}
