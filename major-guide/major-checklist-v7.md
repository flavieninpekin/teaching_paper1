### 7️⃣ 科学研究
> 做学问、探索未知——适合对基础学科有好奇心、不急于就业的学生。

| 专业 | 选科标签 |
|------|----------|
| 数学与应用数学 | 🔓 不限 / 🧲 部分院校 |
| 物理学 | 🧲 物理必选 |
| 化学 | 🧪 化学必选 |
| 生物科学 | 🧬 生物必选 / 🧪 化学建议 |
| 统计学 | 🔓 不限 / 🧲 部分院校 |
| 应用统计学 | 🔓 不限 / 🧲 部分院校 |
| 自然地理与资源环境 | 🧲🧪 物理化学建议 |
| 系统科学 | 🧲 物理建议 |
| 生态学 | 🧪🧬 化学生物建议 |
| 心理学 | 🔓 不限 / 🧬 部分院校 |
| 天文学 | 🧲 物理必选 |

> **关于地理学**：地理学在本科目录中细分为多个方向——**自然地理与资源环境**（侧重地球系统、气候、地貌，归入本章）、**人文地理与城乡规划**（侧重空间规划、人居环境，归入方向⑥）、**地理信息科学**（侧重GIS与遥感，归入方向①）。这种三分法参考了北京大学城市与环境学院的学科组织方式。中学地理教师对应的本科专业见方向⑧。
>
> **关于心理学**：近年**认知与脑科学**（认知心理学×神经科学×计算建模）发展迅速，已成为基础研究的热门交叉领域。应用方向（心理咨询、学校心理）在各专业内部展开说明。

### 1️⃣2️⃣ 农业与生态
> 种地、养动物、搞生物技术、做食品——社会基石，近年被低估的潜力方向。

| 专业 | 选科标签 |
|------|----------|
| 农学 | 🧪🧬 化学生物必选 |
| 动物医学 | 🧪🧬 化学生物必选 |
| 园林 | 🔓 不限 / 🧪🧬 部分院校 |
| 食品科学与工程 | 🧪🧬 化学生物建议 |
| 生物工程 | 🧪🧬 化学生物必选 |

> **生物工程 vs 生物科学 vs 生物医学工程**：生物科学（方向⑦）探索生命原理；生物工程用生物体来生产（酿酒、发酵、生物燃料、酶制剂等），偏工业应用；生物医学工程（方向④）面向医疗设备。三者课程有重叠，但就业方向不同。
