### 7️⃣ 科学研究
> 做学问、探索未知——适合对基础学科有好奇心、不急于就业的学生。

| 专业 | 选科标签 |
|------|----------|
| 数学与应用数学 | 🔓 不限 / 🧲 部分院校 |
| 物理学 | 🧲 物理必选 |
| 化学 | 🧪 化学必选 |
| 生物科学 | 🧬 生物必选 / 🧪 化学建议 |
| 统计学 | 🔓 不限 / 🧲 部分院校 |
| 应用统计学 | 🔓 不限 / 🧲 部分院校 |
| 地理科学 | 🔓 不限 / 🧲🧪 部分院校 |
| 系统科学 | 🧲 物理建议 |
| 生态学 | 🧪🧬 化学生物建议 |
| 心理学 | 🔓 不限 / 🧬 部分院校 |

> **关于心理学**：心理学虽在本科目录中分属理学和应用心理学两个方向，但近年**认知与脑科学**（结合认知心理学、神经科学、计算建模）发展迅速，已成为基础研究的热门交叉领域，故整体归入本章。应用方向（心理咨询、学校心理）在各专业内部展开说明。
>
> **关于认知科学**：认知科学（Cognitive Science）整合心理学、计算机科学、语言学、神经科学，是典型的交叉学科，部分高校已开设相关专业或方向。
