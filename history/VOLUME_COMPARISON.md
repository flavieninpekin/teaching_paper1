# 上下卷写作对比 · 以"春秋战国"一课为例

## 一、正文深度对比（统一主题：尊王攘夷）

| 维度 | 上卷（4-6年级） | 下卷（7-9年级） |
|---|---|---|
| **概念定义** | "尊王攘夷"就是表面上尊重周天子，实际上用天子的名义号令诸侯。 | "尊王攘夷"是在宗法权威衰落、而新权力中心尚未形成的过渡期中出现的一种政治策略——霸主在名分上维持天子权威，在实质上获取征伐会盟的领导权。它的运作依赖于"名"与"实"的分离，是一种典型的制度套利行为。 |
| **因果解释** | 齐桓公为什么要用这套办法？因为如果他直接取代周天子，其他诸侯就会联合起来打他。 | 任何企图直接取代天子的诸侯面临道义谴责和联合讨伐的双重风险。"尊王攘夷"完美避开了这一雷区——它把权力争夺包装成了维护秩序，使齐国在道义和实利两个层面上同时获益。这一策略不仅在春秋有效，九百年后曹操"挟天子以令诸侯"复制了完全相同的逻辑。 |
| **引入的学术概念** | 无。 | "过渡性制度"（transitional institution）：指那些出现在旧秩序崩溃初期、为新体系成型争取时间但自身并不具备长期稳定性的制度形态。霸主政治就是过渡性制度的典型案例。 |
| **史料引用** | 无。 | 《左传·僖公四年》载管仲答楚使全文+译文+解析。 |

## 二、结构设计对比

| 栏目 | 上卷 | 下卷 |
|---|---|---|
| 课内正文 | 故事化叙述（"齐国和楚国打了一仗，齐桓公赢了，于是他当上了霸主……"） | 分析性叙述，每一段都包含一个可识别的论证单元（例如"霸主政治的成因是三方面的……"） |
| 侧边栏 | 史海拾遗（每课1-2个，故事+知识扩展） | 四个独立栏：核心概念（学术术语定义）、史料阅读（原文+译文+解析）、史学方法（因果链搭建/系统分析）、跨文明比较 |
| 课后问答题 | A组6-8题，均为单向知识获取型（"大禹治水用了什么方法？"） | 分AB组：A组5题（复述与要点整理），B组5题（史料分析/比较论证/研究性学习） |
| 论述题 | 无。 | 论证题（600-800字短论文），要求提出核心论点、引用案例、使用反事实推理。 |
| 研究性学习 | 无。 | 四项研究课题（含本地历史调查、对比阅读分析、参考书目分阶标注）。 |

| 题型 | 上卷 | 下卷 |
|---|---|---|
| 概念应用题 | 无。 | 要求用"过渡性制度"分析推恩令/藩镇/预备立宪，400字。 |
| 史料分析题 | 无。 | 提供两段不同立场的史料，要求分析评价差异，并追问"法未败"的证据。 |
| 论证题 | 无。 | 600-800字命题写作，要求含反事实推理。 |
| 比较研究题 | 无。 | 600字以上跨文明比较，需同时处理地理、继承制度、文字、外部威胁、思想传统等变量。 |
| 研究性学习 | 无。 | 本地诸侯国·史记世家阅读笔记（800-1000字）；可选：春秋vs左传体裁对比。 |

## 四、同一知识点的三种表达层次

### 层次一（上卷）：
"商鞅变法让秦国变得很强。他规定士兵在战场上砍一颗敌军的头就可以升一级爵位。所以秦国的士兵打起仗来特别拼命。"

### 层次二（下卷正文）：
"军功爵制打破了世袭贵族对政治权力的垄断：任何出身的人都可以凭军功累积爵位并获取相应的土地和免税特权。这一制度将国家的军事目标与个人的利益最大化动机焊接在了一起。"

### 层次三（下卷核心概念栏）：
"制度化依赖：当一项变革已经嵌入行政日常、社会预期和经济利益分配时，即使最初的推动者消失，既得利益格局也会阻止全面倒退。商鞅被处死后秦法未废，正是因为军功爵制已经成为秦国军队组织的基本制度——废除它会直接动摇国家的军事动员能力。"
