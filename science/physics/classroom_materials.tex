\section*{\textcolor{blue!70!black}{说明}}

本模块对标初中物理课程，系统介绍力、热、光、电、声五大板块的基础知识，含公式推导与例题。适合小学高年级到初中一年级的衔接学习。

\vspace{0.3cm}

\textbf{主要内容（初拟）：}

\begin{enumerate}
  \item \textbf{力学基础}
    \begin{itemize}
      \item 速度、路程与时间：$v = \dfrac{s}{t}$
      \item 重力：$G = mg$
      \item 压强：$p = \dfrac{F}{S}$
      \item 浮力：阿基米德原理 $F_\text{浮} = \rho g V_\text{排}$
    \end{itemize}
  \item \textbf{声与振动}
    \begin{itemize}
      \item 声音的产生与传播
      \item 音调、响度、音色
    \end{itemize}
  \item \textbf{光与光学}
    \begin{itemize}
      \item 光的直线传播、反射定律、折射定律
      \item 凸透镜成像规律
    \end{itemize}
  \item \textbf{热学基础}
    \begin{itemize}
      \item 温度与热量：$Q = cm\Delta t$
      \item 物态变化
    \end{itemize}
  \item \textbf{电学初步}
    \begin{itemize}
      \item 电路与欧姆定律：$I = \dfrac{U}{R}$
      \item 串并联电路
      \item 电功率：$P = UI$
    \end{itemize}
\end{enumerate}
