# 自然科学系列 · 完整大纲

面向北京地区四年级起点的孩子，覆盖初中（中考）和高中（高考3+3选考）的自然科学内容，按 **启蒙→分科入门→深度进阶→拓展探究** 四层递进。总计 **20本**（启蒙1 + 物理6 + 化学4 + 生物4 + 地理2 + 拓展3），约170课。

---

## 学习路径

```
启蒙（1本）
    ↓
物理1·力学入门 → 物理2·声光热 → 物理3·电磁入门
    ↓
化学1·物质构成与化学语言 → 化学2·常见元素与无机反应
生物1·分子与细胞 → 生物2·生物体的结构层次
地理1·自然地理基础 → 地理2·地貌灾害与资源环境
    ↓
物理4·力学与运动定律 → 物理5·曲线能量与动量 → 物理6·电磁波近代
化学3·化学反应原理 → 化学4·有机化学与物质结构
生物3·遗传与进化 → 生物4·生态与生物多样性
    ↓
拓展1·物理竞赛思维  拓展2·化生实验探究  拓展3·地球环境科学
```

## 二、物理系列（6本 · 55课）

### 物理1 · 力学入门（10课）→ `physics/mechanics-intro/`

| 课次 | 标题 | 关键公式 |
|---|---|---|
| 1 | 运动的世界——快与慢 | $v=s/t$ |
| 2 | 力——推和拉的秘密 | 力的三要素、相互作用力 |
| 3 | 重力与弹力 | $G=mg$ |
| 4 | 摩擦力 | 静/滑动/滚动摩擦 |
| 5 | 二力平衡与惯性 | 牛顿第一定律、平衡力vs相互作用力 |
| 6 | 压强 | $p=F/S$ |
| 7 | 液体压强与大气压强 | $p=\rho gh$ |
| 8 | 浮力 | $F_{\text{浮}}=\rho g V_{\text{排}}$ |
| 9 | 简单机械 | $F_1l_1=F_2l_2$ |
| 10 | 功与机械效率 | $W=Fs$, $\eta=W_{\text{有}}/W_{\text{总}}$ |

**对标：** 初中八上+八下（中考计分科目）。每课含定义框、例题框、练一练。

### 物理3 · 电磁入门（7课）→ `physics/electromagnetism-intro/`

**对标：** 初中九年级全册。

|---|---|

| 2 | 速度与加速度（$a=\Delta v/\Delta t$，$v$-$t$图像） |

| 8 | 牛顿第三定律（平衡力vs作用力反作用力，整体法隔离法） |
| 9 | 超重与失重 |

### 物理6 · 电磁、波与近代物理（10课）→ `physics/em-waves-modern/`

| 课次 | 标题 |
|---|---|
| 1 | 静电场（库仑定律、$E=F/q$、电场线） |

| 3 | 电容器与带电粒子运动（$C=Q/U$） |

| 6 | 洛伦兹力（$F=qvB$, $r=mv/(qB)$） |

**对标：** 高中必修3+选择性必修1后半+选择性必修2+选择性必修3精选。

**对标：** 高中选择性必修2+选择性必修3。附录A：元素周期表。
