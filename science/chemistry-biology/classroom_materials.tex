\section*{\textcolor{blue!70!black}{说明}}

本模块将化学与生物的入门知识合并编排。化学部分侧重微观粒子与基本反应，生物部分侧重细胞、人体与生态系统。

\vspace{0.3cm}

\textbf{化学部分（初拟）：}
\begin{enumerate}
  \item 原子与分子——物质是由什么构成的
  \item 元素周期表入门
  \item 化学式与化学方程式：\ch{H2 + O2 -> H2O}
  \item 常见的化学反应类型（化合、分解、置换）
  \item 溶液与浓度
\end{enumerate}

\vspace{0.3cm}

\textbf{生物部分（初拟）：}
\begin{enumerate}
  \item 细胞——生命的基本单位
  \item 植物与动物的分类
  \item 人体八大系统简介
  \item 光合作用与呼吸作用
  \item 生态系统与食物链
\end{enumerate}
