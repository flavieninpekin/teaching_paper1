\chaptertitle{第一讲\quad 行程问题}

\sectiontitle{相遇问题}
\knowbox{相遇路程 $=$ 速度和 $\times$ 相遇时间\\
$S=(v_1+v_2)\times t$}

\begin{exercise}
\item 甲乙两人从相距$240$千米的两地同时出发相向而行，甲每小时行$35$千米，乙每小时行$25$千米，几小时后两人相遇？

\item 客车和货车同时从相距$450$千米的两地出发相向而行，客车每小时行$60$千米，货车每小时行$40$千米，$3$小时后两车相距多少千米？

\item 甲乙两车从相距$360$千米的两地同时出发相向而行，$2.4$小时后相遇。已知甲车每小时行$80$千米，乙车每小时行多少千米？

\item 小明和小红从相距$1.2$千米的两地同时相向而行，小明每分钟走$65$米，小红每分钟走$55$米，$8$分钟后两人相距多少米？
\end{exercise}

\sectiontitle{追及问题}
\knowbox{追及路程 $=$ 速度差 $\times$ 追及时间\\
$S=(v_1-v_2)\times t$}

\begin{exercise}
\item 甲乙两人在相距$12$千米的两地，同时同向出发，甲在前每小时行$5$千米，乙在后每小时行$7$千米，几小时后乙追上甲？

\item 一辆货车以每小时$40$千米的速度从甲地开往乙地，$2$小时后一辆客车以每小时$60$千米的速度也从甲地出发同向而行，几小时后客车追上货车？

\item 小明和小红从学校同时出发去少年宫，小明每分钟走$60$米，小红每分钟走$48$米，$5$分钟后小明到达少年宫，此时小红离少年宫还有多少米？
\end{exercise}

\sectiontitle{流水问题}
\knowbox{顺水速度 $=$ 船速 $+$ 水速\\
逆水速度 $=$ 船速 $-$ 水速\\
船速 $= \dfrac{\text{顺}+\text{逆}}{2}$ \qquad 水速 $= \dfrac{\text{顺}-\text{逆}}{2}$}

\begin{exercise}
\item 一艘船在静水中每小时行$15$千米，水流速度为每小时$3$千米。它顺水航行$72$千米需要几小时？逆水航行$60$千米需要几小时？

\item 一艘船顺水航行$120$千米用了$5$小时，已知水流速度是每小时$2$千米，这艘船逆水航行这段距离需要几小时？

\item 一艘船从A港到B港顺水航行需$6$小时，逆水返回需$8$小时。已知水速为每小时$2$千米，求A、B两港的距离。
\end{exercise}

\sectiontitle{火车过桥问题}
\knowbox{火车过桥路程 $=$ 桥长 $+$ 火车长度\\
火车完全在桥上的路程 $=$ 桥长 $-$ 火车长度}

\begin{exercise}
\item 一列火车长$200$米，以每秒$25$米的速度通过一座长$800$米的大桥，需要多少秒？

\item 一列长$180$米的火车，以每秒$20$米的速度通过一座桥用了$45$秒，这座桥长多少米？

\item 一列火车通过一座长$1200$米的大桥用了$65$秒，以同样的速度通过一座长$1800$米的大桥用了$85$秒，求火车的速度和长度。
\end{exercise}

\chaptertitle{第二讲\quad 工程问题}

\knowbox{工作总量 $=$ 工作效率 $\times$ 工作时间\\
通常把工作总量看作单位"1"。\\
工作效率 $= \dfrac{1}{\text{完成时间}}$}

\begin{exercise}
\item 一项工程，甲单独做要$12$天完成，乙单独做要$18$天完成。两人合作需要多少天完成？

\item 一项工程，甲单独做要$15$天，乙单独做要$10$天，丙单独做要$20$天。三人合作要多少天？

\item 一项工程，甲单独做要$20$天，乙单独做要$30$天。甲先做$5$天后，剩下的两人合作，还要多少天完成？

\item 一项工程，甲单独做需要$10$天，乙单独做需要$15$天。甲先做$2$天后乙加入合作，还需要几天完成？

\item 一项工程，甲乙合作需$12$天完成。甲单独做需$20$天完成，乙单独做需多少天完成？

\item 一项工程，甲单独做$8$天完成，乙单独做$12$天完成。丙单独做$24$天完成。现在先由甲做$2$天，再由乙做$3$天，剩下的丙做，还要几天完成？

\item 水池有甲乙两个进水管和丙一个排水管。单开甲管$3$小时注满，单开乙管$4$小时注满，单开丙管$6$小时排空。三管齐开，几小时能注满水池？
\end{exercise}

\chaptertitle{第三讲\quad 浓度问题}

\knowbox{浓度 $= \dfrac{\text{溶质质量}}{\text{溶液质量}}\times100\%$\\
溶质质量 $=$ 溶液质量 $\times$ 浓度\\
加水稀释：溶质不变，溶液增加。\\
加浓：加溶质或蒸发水。\\
混合：两种溶液的总溶质 $=$ 各自溶质之和。}

\begin{exercise}
\item 在$200$克水中加入$50$克糖，糖水的浓度是多少？

\item 有$300$克$15\%$的盐水，要使其浓度变为$10\%$，需要加水多少克？

\item 有$200$克$10\%$的盐水，要使其浓度变为$20\%$，需要加盐多少克？

\item 有$400$克$12\%$的盐水，要使其浓度变为$20\%$，需要蒸发掉多少克水？

\item 将$20\%$的盐水与$5\%$的盐水混合，配成$15\%$的盐水$600$克，需要$20\%$和$5\%$的盐水各多少克？

\item 有浓度为$25\%$的盐水$200$克，要得到$10\%$的盐水，需要加水多少克？再加入多少克盐能变回$25\%$？
\end{exercise}

\chaptertitle{第四讲\quad 盈亏与鸡兔同笼}

\sectiontitle{盈亏问题}
\knowbox{（盈$+$亏）$\div$（两次分配差）$=$ 人数\\
（大盈$-$小盈）$\div$（两次分配差）$=$ 人数\\
（大亏$-$小亏）$\div$（两次分配差）$=$ 人数}

\begin{exercise}
\item 把一些苹果分给小朋友。每人分$3$个则多$8$个，每人分$5$个则少$12$个。有多少个小朋友？多少个苹果？

\item 把一些铅笔分给同学。每人分$4$支则多$10$支，每人分$6$支则多$2$支。有多少个同学？多少支铅笔？

\item 一组同学去植树。每人植$5$棵则多$3$棵，每人植$6$棵则少$4$棵。有多少同学？多少棵树苗？
\end{exercise}

\sectiontitle{鸡兔同笼}

\begin{exercise}
\item 笼中鸡兔共$35$个头，$94$只脚，鸡兔各几只？

\item 停车场有自行车和三轮车共$12$辆，共有$31$个轮子，自行车和三轮车各几辆？

\item $20$道数学题，做对一道得$5$分，做错一道扣$3$分。小明得了$76$分，他做对了几道？

\item 学校组织$100$名同学去划船，共租了$12$条船。大船每条坐$10$人，小船每条坐$6$人。大船小船各几条？
\end{exercise}

\chaptertitle{第五讲\quad 分数百分数应用}

\begin{exercise}
\item 小明看一本书，第一天看了全书的$\dfrac{1}{5}$，第二天看了全书的$\dfrac{1}{4}$，还剩$55$页没看。全书共多少页？

\item 一根绳子，第一次用去全长的$\dfrac{1}{3}$，第二次用去剩下的$\dfrac{1}{4}$，还剩$6$米，这根绳子原长多少米？

\item 一件商品原价$200$元，先涨价$10\%$，再打九折出售，现价多少元？

\item 一件商品打八折出售便宜了$40$元，原价多少元？

\item 果园里苹果树比梨树多$60$棵，苹果树比梨树多$\dfrac{3}{5}$。梨树和苹果树各多少棵？

\item 水结成冰后体积增加$\dfrac{1}{11}$，冰化成水后体积减少几分之几？

\item 甲数是乙数的$\dfrac{4}{5}$，乙数是丙数的$\dfrac{5}{6}$，甲乙丙三数的和是$210$。甲乙丙各是多少？
\end{exercise}

\chaptertitle{第六讲\quad 比例应用题}

\begin{exercise}
\item 把$240$本按$3:5$分给两个班，每班各得多少本？

\item 一个三角形三个内角之比为$2:3:5$，三个内角各是多少度？这是什么三角形？

\item 甲乙两地的实际距离是$120$千米，在一幅地图上量得$3$厘米。这幅地图的比例尺是多少？在这幅地图上量得A、B两城距离$5$厘米，实际距离多少千米？

\item 一间教室用边长$0.4$米的方砖铺地需要$75$块，如果用边长$0.5$米的方砖需要多少块？

\item 修一段路，全长$1200$米，已修的长度与剩下长度的比是$3:2$，已修了多少米？

\item 一种药水按药粉和水$1:200$配制。要配制$402$千克药水，需要药粉和水各多少千克？
\end{exercise}

\chaptertitle{综合测试}

\begin{exercise}
\item 甲乙两车从相距$480$千米的两地同时出发，相向而行，$3$小时后相遇。甲车每小时行$85$千米，乙车每小时行多少千米？

\item 一项工程甲单独做$15$天完成，乙单独做$10$天完成，丙单独做$30$天完成。三人合作多少天完成？

\item $200$克$10\%$盐水与$300$克$20\%$盐水混合，混合后盐水浓度是多少？

\item 鸡兔共$40$只，脚$110$只，鸡兔各几只？

\item 一本书第一天看全书的$\dfrac{1}{6}$，第二天看剩下的$\dfrac{1}{5}$，还剩$80$页，全书多少页？

\item 比例尺$1:5000000$的地图上，甲城到乙城$4.5$厘米，实际距离多少千米？

\item 甲乙丙三个数的比是$3:4:5$，平均数是$36$，三个数各是多少？
\end{exercise}
