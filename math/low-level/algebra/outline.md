# 《代数——从天平到三位一体》大纲

**写作方法**：同算术教材——先做具体的数字例子，再总结一般规律。
**核心理念**：方程、函数、不等式是同一个"关系"的三种问法。

---

## 开篇：从算术到代数

算术题：`3 + 5 = ?` → 已知算式，算结果。
代数题：`3 + ? = 7` → 已知结果，找那个数。
把 `?` 写成 `x`，就是代数的起点。

---

## 第一章　用字母说话——代数式的诞生（3讲）

**第1讲　从具体规律到字母公式**
- 前算术书已会用字母写运算律（交换律 a+b=b+a）
- 观察数列规律：1,4,9,16 → n²
- 字母的价值：一个式子概括无穷多种情况

**第2讲　代数式的构造与求值**
- 把文字关系翻译成代数式："x的3倍与y的2倍的和"→ 3x+2y
- 代入具体数字算值
- 从基本运算："3个x加2个x"就是5个x → 合并同类项

**第3讲　括号与整式加减**
- 去括号的直观：a+(b-c) 和 a-(b-c) 的实际情境
- 整式加减作为"代数式的整理"

---

## 第二章　寻找未知——等与不等（3讲）

**第4讲　天平与等式**
- 天平平衡 ↔ 等式成立：两边加同样东西依然平衡
- 等式性质（两边同加减同乘除）
- 从"3 + ? = 7"到"x + 3 = 7"——移项的诞生（实质是两边同加减的简写）

**第5讲　解方程与列方程**
- 从一步到多步：2(x+3)-5 = 9
- 去括号再算
- 列方程解应用（数字问题、年龄问题、行程问题）

**第6讲　不等式——从等于到范围**
- 情境："至少多少分才能及格？""不超过多少千克？"
- 不等式性质——与方程不同的关键：乘除负数变号
- 解不等式并在数轴上表示
- **初次暗示三位一体**：y = 2x+1，问 y=7 → 方程，问 y>7 → 不等式
- 不等式的应用

---

## 第三章　两个未知数——从一条线到一个平面（3讲）

**第7讲　方程组——两个条件定两个数**
- 鸡兔同笼的代数版本：x+y=10, 2x+4y=28
- 代入消元、加减消元
- 几何意义：两条直线的交点

**第8讲　不等式组——两个条件同时成立**
- 数轴上的公共区间
- 几何意义：直线上下区域的交集

**第9讲　一次关系三位一体**
- 一次函数 y = kx + b
- 方程：当 y 取定值时求 x（直线与水平线的交点）
- 不等式：当 y 在某个区间时 x 的范围（直线上方或下方）
- 一张图：同一个 y = 2x+1，三种问题一目了然

---

## 第四章　代数式工具箱（3讲）

**第10讲　整式乘法与乘法公式**
- 从竖式到代数：多项式×多项式就是逐项相乘再合并
- 先算大量具体例子，归纳平方差公式 (a+b)(a-b)=a²-b²
- 再归纳完全平方公式 (a±b)²=a²±2ab+b²

**第11讲　因式分解——展开的逆运算**
- 为什么要分解？——把高次方程降次（后一章的伏笔）
- 提公因式、公式法、十字相乘法
- 大量具体例子的对比练习

**第12讲　分式与根式**
- 分式=分数的推广：约分、通分、加减乘除
- 根式=算术平方根的推广：化简、分母有理化
- 分式和根式都是代数式的扩展——"数的扩展"在代数中的对应

---

## 第五章　二次的世界——三位一体升级（3讲）

**第13讲　从一次到二次**
- 实际问题引出二次方程：矩形面积、自由落体
- 因式分解法解二次方程——继承第11讲的成果
- 二次项 ≠ 0 不是技术细节，而是"二次"的定义

**第14讲　配方法与公式法**
- 配方法：把 ax²+bx+c 变成 a(x-h)²+k 的形式
- 从配方法推导求根公式
- 判别式 Δ = b²-4ac：三种情况的几何意义

**第15讲　二次三位一体——抛物线的世界**
- 二次函数 y = ax²+bx+c
- 方程 ax²+bx+c = 0 → 抛物线与 x 轴的交点
- 不等式 ax²+bx+c > 0 → 抛物线在 x 轴上方的部分
- 配方法与顶点的关系
- 一次→二次→下一章的分式、根式：同一个框架不断扩展

---

## 第六章　超越多项式——方程类型的扩张（2讲）

**第16讲　分式方程与根式方程**
- 分式方程：去分母→变整式→解→验根
- 根式方程：移项→平方→解→验根
- 增根的本质：操作扩大了"可能的解"的范围

**第17讲　一张图讲完所有方程类型**
- 整式方程（一次、二次、可化为二次的高次）
- 分式方程
- 根式方程
- 方程组的维度扩展
- 核心策略：化归——全都变成一次或二次

---

## 第七章　复数的诞生——最后一次不够用（2讲）

**第18讲　三次方程的困境**
- 历史背景：16世纪意大利数学竞赛，卡尔达诺的公式
- 三次方程 x³ + px + q = 0 的一般解
- 不可约情况（casus irreducibilis）：明明三个实根，公式里却出现了 √-1
- 邦贝利的发现：√-1 的运算可以自洽
- 不是"x²=-1多无聊"，是"三次方程逼着我们接受 i"

**第19讲　复数运算与复平面**
- i 的运算法则
- 复数的加减乘除、共轭
- 复平面——i 作为新维度
- 代数基本定理：n 次方程恰好有 n 个根（在复数范围内）
- 从自然数到复数的完整数系扩张路线图

总计：7章，19讲

**贯穿线索图**：

```
关系(用代数式描述)
  ├→ 定值问题 → 方程
  ├→ 范围问题 → 不等式
  └→ 输入输出 → 函数
```

每遇到一种新关系类型（线性、一次、二次、分式、根式），都问这三个问题。
