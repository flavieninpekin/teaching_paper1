\chaptertitle{第一章\quad 用字母说话——代数式的诞生}

\sectiontitle{第1讲\quad 从具体规律到字母公式}

\subsection*{算术中已经见过字母了}

翻开《算术》，你早就见过用字母写的东西了：

加法交换律：$a+b=b+a$\\
乘法分配律：$a\times(b+c)=a\times b+a\times c$

这里面的 $a$、$b$、$c$ 不是哪一个具体的数——它们是\textbf{任意数}。一个式子 $a+b=b+a$ 用 26 个字母就概括了"任何两个数相加都可以交换位置"这件事。如果用文字说，要写半天。

\textbf{字母的价值：用一个式子，管无穷多种情况。}

\subsection*{从数列中发现字母公式}

\begin{example}
观察下面几组数，找出规律并用字母写出第 $n$ 个数的表达式：
\end{example}
\begin{solution}
(1) $1,\;4,\;9,\;16,\;25,\;\ldots$\\
这是 $1^2,\;2^2,\;3^2,\;4^2,\;5^2$，所以第 $n$ 个数是 $n^2$。

(2) $3,\;5,\;7,\;9,\;11,\;\ldots$\\
每次加 2。第 1 个是 $3=2\times1+1$，第 2 个是 $5=2\times2+1$，第 3 个是 $7=2\times3+1$，所以第 $n$ 个是 $2n+1$。
\end{solution}

\begin{example}
搭正方形需要多少根小棒？
\end{example}
\begin{solution}
\begin{itemize}
  \item 1个正方形：4根
  \item 2个正方形（连在一起）：7根（因为共用1条边）
  \item 3个正方形：10根
  \item 4个正方形：13根
\end{itemize}
每多一个正方形，多用 3 根小棒。\\
第 1 个：$4 = 3\times1+1$\\
第 2 个：$7 = 3\times2+1$\\
第 $n$ 个：$3n+1$ 根。
\end{solution}

\begin{example}
一个长方形的长是 $a$，宽是 $b$。用字母写出它的周长和面积。
\end{example}
\begin{solution}
周长 = $2a+2b$（两条长加两条宽），\\
面积 = $a\times b$（通常省略乘号写成 $ab$）。
\end{solution}

\subsection*{乘号的省略}

在代数里，乘号经常被省略以节省书写：
\begin{itemize}
  \item $a\times b$ 写成 $ab$
  \item $2\times a$ 写成 $2a$
  \item $x\times y\times z$ 写成 $xyz$
\end{itemize}
数字写在字母前面，$2a$ 表示 $2\times a$，不是 $2+a$。

\sectiontitle{第2讲\quad 代数式的构造与求值}

\subsection*{把文字翻译成代数式}

代数式的核心能力：把一段话变成一串符号。

\begin{example}
把下面的话翻译成代数式：
\end{example}
\begin{solution}
"$x$ 的 3 倍与 $y$ 的 2 倍的和" → $3x + 2y$

"$a$ 与 $b$ 的差的平方" → $(a-b)^2$

"$m$ 的 $\dfrac{3}{4}$ 比 $n$ 多 5" → $\dfrac{3}{4}m = n + 5$ 或 $\dfrac{3}{4}m - 5 = n$
\end{solution}

\begin{example}
小明今年 $x$ 岁，爸爸比小明大 28 岁。用代数式表示：
\end{example}
\begin{solution}
(1) 爸爸的年龄：$x + 28$\\
(2) 5 年后小明的年龄：$x + 5$\\
(3) 5 年后爸爸的年龄：$(x + 28) + 5 = x + 33$\\
(4) 10 年前爸爸的年龄：$(x + 28) - 10 = x + 18$
\end{solution}

\subsection*{代数式的值}

代入具体的数，就能算出代数式的值。

\begin{example}
当 $a=2$，$b=3$ 时，求 $3a + 2b$ 的值。
\end{example}
\begin{solution}
把 $a$ 换成 2，$b$ 换成 3：
$3a + 2b = 3\times2 + 2\times3 = 6 + 6 = 12$。
\end{solution}

\begin{example}
当 $x=5$ 时，求 $2x^2 - 3x + 1$ 的值。
\end{example}
\begin{solution}
$2x^2 - 3x + 1 = 2\times5^2 - 3\times5 + 1 = 2\times25 - 15 + 1 = 50 - 15 + 1 = 36$。
\end{solution}

\subsection*{合并同类项——整式的整理}

先看这个：

$3x + 2x = \;\boxed{\;}\;?$

"3 个 $x$ 加 2 个 $x$"就是 5 个 $x$，所以 $3x + 2x = 5x$。这个过程叫\textbf{合并同类项}。

什么叫同类项？\textbf{字母相同、字母的指数也相同的项}。

\begin{example}
合并同类项：$4a + 3b - 2a + b$
\end{example}
\begin{solution}
先找同类项：\\
$a$ 类：$4a$ 和 $-2a$ → $4a-2a = 2a$\\
$b$ 类：$3b$ 和 $+b$ → $3b+b = 4b$\\
所以原式 $= 2a + 4b$。
\end{solution}

\begin{example}
合并同类项：$5x^2 + 3x - 2x^2 + x - 1$
\end{example}
\begin{solution}
$x^2$ 类：$5x^2 - 2x^2 = 3x^2$\\
$x$ 类：$3x + x = 4x$\\
常数类：$-1$ 单独\\
结果：$3x^2 + 4x - 1$。
\end{solution}

\sectiontitle{第3讲\quad 括号与整式加减}

\subsection*{去括号为什么这样去？}

先看数字的例子：

$10 + (3 + 2) = 10 + 5 = 15$，$10 + 3 + 2 = 15$。\\
所以 $a + (b + c) = a + b + c$。

$10 - (3 + 2) = 10 - 5 = 5$，$10 - 3 - 2 = 5$。\\
所以 $a - (b + c) = a - b - c$。

\textbf{括号前面是加号，去括号不变号；括号前面是减号，去括号全变号。}

\begin{example}
去括号：$(3x + 2y) + (x - y)$
\end{example}
\begin{solution}
原式 $= 3x + 2y + x - y = 4x + y$。
\end{solution}

\begin{example}
去括号：$(4a - 3b) - (2a + b)$
\end{example}
\begin{solution}
原式 $= 4a - 3b - 2a - b = 2a - 4b$。
注意：减号后面的括号去掉时，里面的每一项都要变号。
\end{solution}

\subsection*{整式加减——几个例子就够了}

整式加减就是：先去括号，再合并同类项。

\begin{example}
计算：$(3x^2 - 2xy + y^2) - 2(x^2 - xy + 2y^2)$
\end{example}
\begin{solution}
原式 $= 3x^2 - 2xy + y^2 - 2x^2 + 2xy - 4y^2$\\
$= (3x^2 - 2x^2) + (-2xy + 2xy) + (y^2 - 4y^2)$\\
$= x^2 - 3y^2$。
\end{solution}

\begin{example}
计算：$(5a + 3b - 2c) - (2a - 4b + c) + (-a + b - 3c)$
\end{example}
\begin{solution}
原式 $= 5a + 3b - 2c - 2a + 4b - c - a + b - 3c$\\
$a$ 类：$5a - 2a - a = 2a$\\
$b$ 类：$3b + 4b + b = 8b$\\
$c$ 类：$-2c - c - 3c = -6c$\\
结果：$2a + 8b - 6c$。
\end{solution}
