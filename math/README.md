# 数学模块 · 整体大纲

> 面向北京地区四年级起点的课外拓展学习材料，兼顾课内衔接与数学竞赛方向。
> 核心目标：六年级结束通过 2+4 选拔 → 初中竞赛深化 → 高联二试进阶。

---

## 一、设计思路

### 分段
- **第一段：低学段（G4~初三）**——主攻2+4选拔，学完初中课内+竞赛基础
- **第二段：高学段（高中）**——2+4贯通培养，覆盖高中课内+高联预备

### 三线并行
每条线内部可拆分为多册，每册独立成书：

```
┌─ 计算/应用专项（独立练习册，贯穿低学段）
├─ 知识精讲（代数 / 几何 / 数论 / 组合 / 微积分 / 解析几何 等）
└─ 竞赛专题（低学段侧重竞赛入门，高学段侧重高联二试）
```

### 样式
所有文档统一使用 `teaching-style.sty` 样式包，XeLaTeX 编译。
插图策略：能用 TikZ 绘制的几何图直接嵌入，复杂场景图写需求文件到 `figure/requirements/`。

## 三、各模块内容概览

### 低学段·主线（7模块）

| 模块 | 书名 | 核心内容 |
|------|------|---------|
| 算术 | 《算术》 | 整数/小数/分数四则运算 → 有理数混合运算、绝对值、平方根/立方根、实数 |
| 代数 | 《代数上册》 | 一元一次/二次方程、方程组、不等式、含参讨论、绝对值不等式 |
| 代数 | 《代数下册》 | 整式运算、乘法公式、因式分解全方法、分式与根式、函数（正反比例/一次/二次）、复数入门 |
| 几何 | 《几何上册》 | 角度、三角形、全等、四边形、相似、圆（垂径/圆周角/切线/弧长扇形） |
| 几何 | 《几何下册》 | 锐角三角比、解直角三角形、正弦/余弦定理、向量初步、几何变换、综合模型 |
| 数论 | 《数论基础》 | 整除特征、质因分解、GCD/LCM、同余、不定方程、进位制、完全平方数 |
| 组合 | 《组合》 | 集合与逻辑、加乘原理、排列组合、容斥原理、抽屉原理、图论（一笔画/树/最小生成树） |
| 统计 | 《统计与概率初步》 | 数据整理、统计图表、平均数/中位数/众数、方差、古典概型、频率与概率 |

### 低学段·练习册（3册）

| 书名 | 定位 | 适用 |
|------|------|------|
| 《算术计算基本功》 | 四则运算纯练习（整数/小数/分数） | 4~5年级 |
| 《代数式计算专项》 | 整式/乘法公式/因式分解/分式/根式练习 | 5~初一 |
| 《应用题专练》 | 分类专题训练（行程/工程/浓度/盈亏/鸡兔同笼/分数百分数/比例） | 4~6年级 |

### 高学段（6册）

| 模块 | 书名 | 核心内容 |
|------|------|---------|
| 代数 | 《代数》 | 函数概念与性质、基本初等函数、三角恒等变换、数列（等差/等比/递推）、不等式证明、均值与柯西不等式、数学归纳法、复数三角形式、向量坐标与数量积 |
| 微积分 | 《微积分初步》 | （必修）极限、导数（含隐函数/参数/相关变化率）、洛必达与不等式证明、泰勒展开、不定积分（含有理函数/三角积分）、定积分（含反常积分/积分不等式/黎曼和求极限）、ODE（含伯努利/二阶常系数） |
| 微积分 | 《多元微积分》 | （扩展/物理竞赛）偏导与全微分、梯度与方向导数、重积分（柱/球坐标）、曲线/曲面积分、梯度·散度·旋度、格林/高斯/斯托克斯定理 |
| 解析几何 | 《平面解析几何》 | 直线方程及位置关系、圆、椭圆/双曲线/抛物线定义与性质、直线与圆锥曲线、轨迹方程、极坐标与参数方程 |
| 解析几何 | 《线性代数初步》 | （重写版）从平面几何变换出发，矩阵=变换的代码，行列式=面积放大倍率（含线性相关几何意义），特征=不变方向（含施密特正交化），齐次坐标与仿射变换，二次型与圆锥曲线分类（主轴定理代替联立韦达） |
| 立体几何 | 《立体几何》 | 公理体系、平行/垂直判定与性质、空间角与距离、向量法（坐标法）、多面体与旋转体体积/表面积、欧拉定理、截面与展开图 |
| 概率统计 | 《概率论与数理统计》 | 排列组合与二项式定理、条件概率与贝叶斯、随机变量与分布列、二项/超几何/正态分布、期望与方差、大数定律★、中心极限定理★、参数估计★、假设检验★、回归分析★ |

> ★ 标记为高考不考内容，教学中可酌情跳过。

## 四、学习路线图（时间轴）

```
四年级 ──────────────────────────────────────────
   算术（整数加减乘除简算）
   代数上册（方程）
   几何上册（角度/三角形）
   练习册：应用题专练（行程/工程）

五年级 ──────────────────────────────────────────
   算术（小数/分数 → 有理数/绝对值）
   代数上册（不等式）→ 代数下册（因式分解/函数）
   几何上册（全等/四边形/相似/圆）
   数论基础
   组合（集合/排列组合/抽屉）
   练习册：算术计算基本功 + 代数式计算专项
   练习册：应用题专练（浓度/盈亏/鸡兔同笼）

六年级（2+4选拔冲刺）─────────────────────────
   算术（平方根/实数）
   代数下册（二次函数/复数入门）
   几何下册（解三角/向量）
   数论（同余/不定方程）
   练习册：代数式计算专项 + 应用题（分数百分数/比例）
   选拔真题模拟

初一~初三（2+4贯通 / 继续深化）────────────────
   代数下册深化（函数/复数）
   几何下册深化（正余弦定理）
   组合（图论）
   统计与概率初步
   练习册：代数式计算专项

高一~高三（高学段）───────────────────────────
  代数（函数/数列/不等式/复数/向量）
  微积分（导数/积分/微分方程）
  平面解析几何
  线性代数初步
  立体几何
  概率论与数理统计
```
