# 《微积分初步》深化方案

> 定位不变：高学段必修、高联有用。深度上增加到"竞赛运算熟练度"级别。
> 不是加理论（不搞 $\varepsilon$-$\delta$），而是加**技巧密度和应用广度**。

## 各章深化点

### 第1章 极限

| 原内容 | 加深方向 |
|--------|---------|
| 两个重要极限 | 补充：$\lim_{x\to0}\frac{a^x-1}{x}=\ln a$，$\lim_{x\to0}\frac{(1+x)^\alpha-1}{x}=\alpha$——竞赛中频繁出现 |
| 极限运算法则 | 增加无穷小阶的比较、等价无穷小替换的高级技巧 |
| | 新加：夹逼准则求极限（高联常见） |

### 第2章 导数

| 原内容 | 加深方向 |
|--------|---------|
| 基本公式 | 补充：反三角函数导数、双曲函数导数简介 |
| 链式法则 | 增加隐函数二阶求导、参数方程求导 |
| | 新加：**相关变化率**（related rates）专题——竞赛物理通用 |
| | 新加：**高阶导数的莱布尼茨公式**——用于展开和证明 |

### 第3章 导数的应用

| 原内容 | 加深方向 |
|--------|---------|
| 单调/极值/最值 | 增加含参讨论、隐函数极值 |
| 洛必达法则 | 增加 $0\cdot\infty$、$\infty-\infty$、$1^\infty$ 等不定型转化技巧 |
| 函数作图 | 增加渐近线（斜渐近线）的求法 |
| | 新加：**不等式证明**（利用导数构造辅助函数）——高联大题高频考点 |
| | 新加：**方程根的个数讨论**（参数分离+导数分析） |

### 第4章 泰勒展开

| 原内容 | 加深方向 |
|--------|---------|
| 麦克劳林展开 | 增加展开的误差估计（拉格朗日余项） |
| | 新加：**用泰勒展开求极限**（比洛必达更强大的工具） |
| | 新加：**用泰勒展开证明不等式**（放缩法） |

### 第5章 不定积分

| 原内容 | 加深方向 |
|--------|---------|
| 换元/分部 | 增加**有理函数积分**（部分分式展开） |
| | 增加**三角函数积分**的常见技巧（万能公式、$\sin^2x$ 和 $\cos^2x$ 的递推） |
| | 增加**倒代换**、**欧拉代换**简介 |

### 第6章 定积分

| 原内容 | 加深方向 |
|--------|---------|
| 计算/应用 | 增加**反常积分**（无穷限/无界函数）的审敛与计算 |
| | 增加**旋转体侧面积**、**旋转体的体积**（柱壳法） |
| | 新加：**积分不等式**（积分形式的柯西不等式、估值定理） |
| | 新加：**定积分求极限**（黎曼和定义倒用）——高联竞赛技巧 |

### 第7章 常微分方程

| 原内容 | 加深方向 |
|--------|---------|
| 可分离/一阶线性 | 增加**伯努利方程**（换元化为一阶线性） |
| | 增加**二阶常系数线性齐次方程**（特征根法） |
| | 增加**可降阶的微分方程**（$y''=f(x)$ / 不含$y$型 / 不含$x$型） |
