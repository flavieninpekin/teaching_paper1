# 社科系列整体规划

## 定位

社会科学系列面向四年级起点学生，对标北京中考/高考文科综合，覆盖历史、政治、经济、哲学、法学五大领域。核心理念：**用故事引发兴趣，用逻辑建立框架**。

## 模块依赖关系

```
G4 ──┬── 古代中国 ──→ 古代世界 ──→ 近现代史 ──→ 经济地理
     ├── 经济与社会（故事式启蒙）
     ├── 道德与法治
     └── 哲学初步 ──→ 政治学与法学入门
```
