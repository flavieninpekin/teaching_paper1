# 社科系列整体规划

## 定位

社会科学系列面向四年级起点学生，对标北京中考/高考文科综合，覆盖经济、政治、法律、哲学、伦理五大领域。核心理念：**用故事引发兴趣，用逻辑建立框架**。

> **历史系列已拆分为独立模块**，详见 `history/HISTORY_CATALOG.md`。历史部分包括古代中国、古代世界/欧洲史、近现代史三个子模块，保持原有学习路径。

## 模块依赖关系

```
G4 ──┬── society-intro（社科启蒙）──→ 经济与社会
     ├── 道德与法治
     └── 哲学初步 ──→ 政治学与法学入门 ──→ 经济地理
```
